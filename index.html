The Hover of Lies

Bab 1: Kode yang Tak Pernah Terbang

Rahmat menatap monitor dengan keyakinan penuh. Barisan kode di index.php sudah diperbaiki. Warnanya lebih terang, tombolnya lebih hoverable, dan hidupnya seolah lebih bermakna. Ia menekan Ctrl + S lalu dengan bangga berkata:

"Sudah, sekarang pasti tampilannya berubah..." katanya, tak tahu bahwa malapetaka telah dimulai.

Di balik layar, VS Code tersenyum licik. "Sudah kusimpan... di hatimu saja, bukan di servermu," bisiknya.

File Manager muncul bak detektif tua, memeriksa direktori /home/hrosace/. Ia mengangkat alis: "Mana file index.php yang terbaru? Ini masih versi jadul, Bro."

Cmd yang mendengar kabar itu langsung teriak: "dir /s index.php" tapi tak menemukan bukti kuat. Semua hanya bayangan.

Di pojok ruangan, Cache tertawa sambil merokok. "Lihat? Semua percaya padaku. Mereka takkan pernah tahu file lama masih kupajang di browser."

Rahmat mulai curiga. Ditatapnya browser. Dihover-nya tombol yang harusnya berubah warna. Tidak ada yang terjadi. Cuma efek placebo.
