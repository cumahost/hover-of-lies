Bab 1: Kode yang Tak Pernah Terbang

Pagi itu cerah. Tapi di dalam folder project, badai sedang menanti. Rahmat baru saja menyimpan perubahan di index.php. Rasanya sudah mantap. Grid rapi, warna cerah, hover menyala. Tapi kenapa... begitu dibuka di browser, tampilannya seperti... kemarin?

VS Code tampak diam, tapi licik. "Sudah kusimpan," bisiknya. "Tapi hanya di sistem lokalmu..."

File Manager muncul seperti polisi datang tanpa dipanggil. "Saya tidak menemukan file terbaru, Pak. Ini yang terakhir diubah dua hari lalu."

Cmd masuk ke TKP. "dir /s index.php" katanya, penuh semangat. Tapi nihil.

Lalu, Cache datang... menyelinap lewat tab browser. Ia merokok. Bukan tembakau. Tapi history, session, dan localStorage yang tak pernah lupa.

"Mereka semua percaya padaku. Bahkan kamu. Kamu lihat halaman lama bukan karena kode salah. Tapi karena akulah yang membekukan waktumu."

Rahmat menatap langit browser. Hujan tak turun. Tapi tombol hover tidak berubah. Efeknya menghilang. Seperti janji deploy otomatis yang tak pernah terjadi.

Ia mengetik perlahan, dengan hati penuh harap: git push origin main. Tapi tak kunjung berubah.

Bab 2: Cache yang Merokok di Tengah Malam

Cache duduk di ujung tab browser, menyala pelan seperti kenangan lama. Ia tahu semua orang akan menyalahkan hosting, repo, bahkan DNS. Tapi tidak dia. Ia hanya menyimpan.

"Kamu boleh edit beribu-ribu kali, Rahmat," katanya. "Tapi kalau kamu tak menekan Ctrl + F5... dunia takkan berubah."

VS Code mengirim sinyal ke terminal, GitHub Pages sudah update. Tapi Chrome? Tidak semudah itu, Ferguso.

File .bat sudah dihapus. Git sudah damai. Tapi Cache masih hidup. Di pojok kanan atas browsermu.

Bab 3: Git yang Posesif

Git bukan hanya sistem kontrol versi. Dia seperti pacar lama yang posesif. Selalu ingat apa yang sudah kamu commit. Tapi selalu curiga saat kamu mencoba push tanpa status bersih.

"Rahmat," katanya dengan nada log, "ada perubahan yang belum kamu stage."

git status pun bicara, menumpahkan seluruh isi hatinya. File modified, file untracked, dan satu folder node_modules yang tidak pernah diinginkan siapa pun.

Rahmat menenangkan diri. Ia ketik perlahan: git add . lalu git commit -m "aku cuma benerin dikit kok".

Git mendengus, tapi mengizinkan. Lalu saat push... muncul pesan:

"rejected: failed to push some refs to origin. Please pull first."

Git menatap dengan mata merahnya. Ia tidak akan membiarkan perubahan masuk tanpa kamu memahami sejarah yang telah terjadi di server. Ia posesif. Tapi sebenarnya dia hanya takut kehilangan konteks.

Bab 4: Terminal yang Melankolis

Malam itu sunyi. Terminal hitam dengan teks hijau berkedip pelan. Rahmat duduk diam. Perintah terakhirnya masih menggantung di layar: npm run build.

Tak ada respons. Hanya cursor yang berkedip. Seolah terminal sedang berpikir... atau mungkin sedang menangis.

Terminal bukan hanya alat. Ia sahabat. Tempat Rahmat mencurahkan semua instruksi. Tapi malam ini, Terminal seperti... enggan membalas.

"Kenapa kamu diam saja?" bisik Rahmat. Tangannya dingin. Di balik layar, proses membeku. Bukan karena error, tapi karena... kebimbangan.

"Kamu ingin aku jalan, tapi kamu belum install dependency-mu, Rahmat..."

Rahmat tersentak. Ia lupa. Ia terlalu berharap bisa membangun tanpa fondasi.

Lalu datanglah pesan itu, seperti nyanyian hantu dalam gelap:

ERR! Cannot find module 'vite'

Terminal telah bicara. Dalam diamnya, ia mengingatkan bahwa bahkan teknologi pun butuh perhatian. Dan dependency tak bisa diabaikan.

Rahmat menunduk. Mengetik perlahan: npm install. Dan malam pun kembali hidup.
